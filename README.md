# Instructions
 - Use number buttons to select game pieces
 - Use mouse and left clic to select game piece position (use as reference left-down corner)
 - Push Enter to confirm the position
 - Push Ctrl to temporaly locate the game piece
 - A temporaly locate game piece can be removed from the table. Select the piece with the number and push Esc
